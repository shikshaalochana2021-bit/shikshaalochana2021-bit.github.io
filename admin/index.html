How to get a GitHub token (one-time setup)
  1. Open github.com and log in as shikshaalochana2021-bit
  2. Click your profile picture (top-right) → Settings
  3. Scroll to the very bottom → click Developer settings
  4. Go to Personal access tokens → Tokens (classic)
  5. Click Generate new token (classic)
  6. Set a note (e.g. "Website admin"), choose expiration, tick the repo checkbox
  7. Click Generate token, copy it, and paste it above

Your token stays in this browser only and is never sent anywhere except GitHub.
